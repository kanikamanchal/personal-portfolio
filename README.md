# Personal Portfolio Website

## Overview

This is my personal portfolio website developed to showcase my skills, projects, education, and achievements. The website is designed with a clean and responsive layout to provide a smooth user experience across different devices.

## Features

* Responsive design for desktop, tablet, and mobile devices
* Modern and user-friendly interface
* About Me section
* Skills showcase
* Projects portfolio
* Contact information section
* Smooth navigation and interactive elements

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Bootstrap

## Purpose

The goal of this project is to present my web development skills, highlight my projects, and create an online professional presence.
